\texttt{*   **Enhanced UX:** Provides "presence awareness," allowing users to see who is active and where, facilitating smoother co{-}authoring.} \\
\texttt{*   **Conflict Resolution:** Helps prevent multiple users from editing the same section simultaneously by highlighting active cursors.} \\
\texttt{*   **Performance:** By tracking presence separately from the heavy document diffing logic, we can optimize the main editing engine's performance while adding a lightweight, high{-}frequency update layer.} \\
