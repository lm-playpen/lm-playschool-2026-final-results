\texttt{The argument structure is:} \\
\texttt{1.  \$P \textbackslash{}rightarrow (T \textbackslash{}lor M)\$} \\
\texttt{2.  \$T \textbackslash{}rightarrow \textbackslash{}neg P\$} \\
\texttt{3.  Therefore, \$\textbackslash{}neg P \textbackslash{}rightarrow M\$} \\
